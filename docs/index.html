BareScript

Lightweight userscript manager for Chrome.
Simple. Fast. No bloat.

Add to Chrome

Simple Scripts

Create custom JavaScript that runs on specific websites using @match patterns.

Clean Dashboard

Manage all your scripts from a simple, fast dashboard with syntax highlighting.

Privacy First

No tracking, no cloud sync. Your scripts stay on your device.

Dark Mode

Automatic dark mode support that follows your system preference.

Example Script

// ==BareScript==
// @name        Example
// @match       https://example.com/*
// @run-at      document-end
// ==/BareScript==

document.body.style.background = '#1a1a1a';
document.body.style.color = '#ffffff';
