# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added

- Library support - create reusable code modules with `@type library`
- ES module imports - use `import` to include libraries in scripts
- SPA navigation detection - scripts re-run on pushState/replaceState

### Changed

- Scripts are no longer injected into the DOM
